Gitの勉強の為にコマンドのチートシート？を書くよ！

git init [--bare] [directory]

git init
カレントディレクトリをgitの作業用リポジトリにする。
具体的には、.gitディレクトリを作る。
[directory]を指定すると、指定したディレクトリに.gitディレクトリを作り、
gitリポジトリ化する。
git init --bare
カレントディレクトリをgitのリモート用(共用)リポジトリにする。
git init コマンドで作られる.gitディレクトリの中身がカレントディレクトリ内に展開される。
git init でできる作業用リポジトリとの違いは、pushされたときにファイルが即最新のコミットの状態になること。
